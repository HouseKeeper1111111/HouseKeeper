1. Формат коммит месседжа
2. Пишется на английском
3. Заголовок не более 50 символов
4. Если в заголовок не помещается информация, которую нужно передать в коммит месседже, то добавляется описание.
5. Заголовок и описание каждый начинаются с прописной буквы
6. Между заголовком и описанием должна быть пробельная строка
7. Должны быть вручную проставлены переносы строк описания, чтобы оно было читабельным.
8. Месседж должен сообщать о действиях в прошедшем времени и отвечать на вопрос “Что было сделано?“

Пример:

Reminder done

Entitie of reminder created. Made possibility to set the response time.
Made possibility to choose reminder type. Made possibility to choose
reminder cathegory.
